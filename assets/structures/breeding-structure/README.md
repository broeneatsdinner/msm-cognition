# Breeding Structure assets

This directory is intended to mirror `assets/eggs/` for Breeding Structure detection.

Run `bin/download-breeding-structure-assets.sh` from the root of this unpacked kit to download the current source images into this directory.

Expected files:

```text
normal-breeding-structure.webp
enhanced-breeding-structure.webp
locked-breeding-structure.webp
paironormal-breeding-structure.webp
paironormal-enhanced-breeding-structure.webp
paironormal-locked-breeding-structure.webp
```

These images come from the My Singing Monsters Wiki / Fandom Breeding Structure page. Keep source attribution with the files if committing them into a public repository.
