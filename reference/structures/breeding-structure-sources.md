# Breeding Structure source images

Source page:

```text
https://mysingingmonsters.fandom.com/wiki/Breeding_Structure
```

Source notes:

- The Breeding Structure is movable.
- The structure size is listed as 4 x 4.
- The structure limit is listed as 2.
- The Enhanced Breeding Structure has the same size and shape as the normal one, with visual changes.
- Paironormal Carnival has a unique design.
- The detector should eventually search the full screenshot for zero, one, or two candidate Breeding Structures rather than relying on fixed coordinates.
