# Data

Raw Google Sheets exports for MSM Cognition.

These files are snapshots of the uploaded Google Sheets workbook. They are intentionally kept close to the spreadsheet shape so later scripts can normalize them into structured rule data.

## Files

- `raw/islands_msm.csv` / `raw/islands_msm.json`
- `raw/monsters_msm.csv` / `raw/monsters_msm.json`
- `raw/wublin_blueprints_msm.csv` / `raw/wublin_blueprints_msm.json`

The `docs/data/` directory contains Markdown views intended for GitHub browsing.
